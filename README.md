# Backendless documentation

### Installation
1. Install python 3+
2. Install pip
3. `pip install mkdocs`
4. `pip install mkdocs-material`

### Build
1. Inside documentation folder for examole in `backendless-pro-manager` execute `mkdocs serve` the docs will be available by the follwoing address http://127.0.0.1:8000/
2. To create production build execute: mkdocs build

### Additional information
1. https://squidfunk.github.io/mkdocs-material/reference/
2. https://squidfunk.github.io/mkdocs-material/plugins/
