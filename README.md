# Backendless documentation

### Installation
1. Install python 3+
2. Install pip
3. `pip install mkdocs`
4. `pip install mkdocs-material`
5. `pip install mkdocs-include-markdown-plugin`
6. `pip3 install mkdocs-glightbox`

### Build
1. Inside the documentation folder, for example in `automations`, execute `mkdocs serve`. Ð¢he docs will be available at the following address http://127.0.0.1:8000/
2. To create production build execute: `mkdocs build`
