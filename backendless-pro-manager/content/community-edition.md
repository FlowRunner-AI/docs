# Backendless Pro Community Edition - Single Node Installation

Backendless Pro Community Edition is designed to be deployed within a Kubernetes environment. This particular version emphasizes a single node installation, putting all pieces of the Backendless ecosystem on one unit for efficiency and ease of access.

Whether you're a seasoned developer or just getting started, Backendless Pro Community Edition provides the tools and support you need to succeed. Dive into the documentation, explore the features, and start building your next serverless application with Backendless Pro Community Edition today!

### Dependencies
Backendless Pro Community Edition relies on the following components within the Kubernetes environment:

 - **MySQL** 8+ database instance installed as `bl-mysql`.
 - **Redis** 6+ caching server installed as `bl-redis`.
 - **MongoDB** 5+ database instance installed as `bl-mongo`.
 - **The file system** required for Backendless is mounted from the host machine into the Backendless workloads within Kubernetes, enabling access to necessary files and directories.
