### Backendless Pro Dependencies

Running Backendless Pro Community Edition in Kubernetes requires:

- **MySQL (8+)**: Installed as `bl-mysql`.
- **Redis (6+)**: Installed as `bl-redis`.
- **MongoDB (5+)**: Installed as `bl-mongo`.
- **File System Access**: The necessary file system is mounted from the host machine into Kubernetes, allowing access to needed files and directories.
